spothopper-ios
==============

## Setup

1. `gem install cocoapods`
2. Clone repository - `git clone git@github.com:RokkinCat/spothopper-ios.git`
2. `cd spothopper-ios`
3. CocoaPods
  1. Install CocoaPods gem - `gem install cocoapods`
  2. Install all pods - `pod install`
4. Open `SpotHopper.xcworkspace`
